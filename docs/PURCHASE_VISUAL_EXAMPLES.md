## Key Takeaways

1. **Same cost = Merge**: Items with matching cost consolidate automatically
2. **Different cost = Variant**: Items with different cost get unique names
3. **Clear naming**: Variants use _1, _2, _3 pattern
4. **User feedback**: Clear messages explain what happened
5. **No data loss**: All items preserved, just organized differently
